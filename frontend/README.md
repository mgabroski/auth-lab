# Frontend Folder Map

This folder contains the tenant-facing frontend application.

## Read in this order for frontend work

1. `../AGENTS.md`
2. `AGENTS.md`
3. `src/shared/engineering-rules.md`
4. `docs/module-skeleton.md`
5. relevant backend API docs

## What lives here

- `AGENTS.md` -> frontend-specific AI/review router
- `docs/module-skeleton.md` -> standard frontend module shape
- `src/shared/engineering-rules.md` -> frontend implementation law
- route, page, component, and shared UI code -> actual frontend implementation
- tests -> frontend proof at the appropriate level

## Important rule

Do not treat this file as a second router, a product brief, or an architecture doc.
It is only a folder map.
